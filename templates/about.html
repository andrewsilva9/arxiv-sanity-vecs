A lighter-weight arxiv-sanity from-scratch re-write. Periodically polls arxiv API for new papers. Then allows users to like/dislike papers, and recommends similar papers based on sequence embeddings for abstracts/titles. You can also search for similar papers using sequence embeddings. Hopefully I will clean up the UI a bit in the future, to make it clearer when you've liked/disliked papers, and to let you view previously liked/disliked papers.

This particular instance indexes papers from cs.CV, cs.LG, cs.CL, cs.AI, cs.NE, cs.RO, and only since early/mid-2023.

The code for this website is adapted from the open sourced code under arxiv-sanity-lite.

Adapted while procrastinating thesis-defense prep as a labor of anxiety @andrewsilva9. Originally developed in spare time as a labor of love by @karpathy.
